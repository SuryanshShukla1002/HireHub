# Job Posting App

A simple CRUD-based Job Posting application where users can view job listings and employers can post new jobs.

## Features
1. Job Postings (Home Page)

### Displays all available job postings

#### Each job card shows:

- Job title
- Company name
- Location
- Job type
- Actions:
- See Details – View full job information
- Delete – Remove a job posting instantly

## Search jobs by job title

#### 2. Job Details Page

- Job title
- Company name
- Location
- Salary
- Job type
-Job description

### Required qualifications (ordered list)

#### 3. Post a Job

- Job title
- Company name
- Location
- Salary
- Job type (dropdown)
- Job description
- Required qualifications
- Form validation:
- All fields are required
- Proper placeholders provided

#### 4. Responsive Design

- Fully responsive layout

Works smoothly on desktop, tablet, and mobile devices

Tech Stack

Frontend: React, Bootstrap

Backend: Node.js, Express

Database: MongoDB
